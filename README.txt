Proje Özeti
Bu proje, ASP.NET Core MVC kullanılarak geliştirilmiş bir Personel Yönetim Sistemi'dir.
Projede personel bilgileri, maaş, departman gibi veriler SQL veritabanında saklanır.
Kullanıcı, personel ekleme, düzenleme, silme işlemleri ile birlikte maaş bilgilerini de yönetebilir.


Geliştirme Ortamı
-Visual Studio 2022

-ASP.NET Core 8.0

-Entity Framework Core

-SQL Server (Azure destekli)

-.NET 8 SDK

Proje İçeriği (Kısaca)

-Models/ – Veritabanı tablolarını temsil eden sınıflar

-Controllers/ – HTTP isteklerini yöneten denetleyiciler

-Views/ – Razor sayfaları (UI)

-Data/ – Veritabanı bağlantısı ve ilk veri migrasyonları
